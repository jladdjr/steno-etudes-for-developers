# Etude 10: Command Line Lingo

```
Start a new session.

Run the script and check the return code.
Pipe the output to stdout, to stderr, to the file, to the other process.

Source the script so the environment variables are defined in your current session.

It's a utility, command, binary, executable
that uses an option, flag, parameter, argument.

It has a man page, documentation, usage examples.

Log in with your password.

Your prompt shows your username.
It's to the right of your cursor.

The file listing shows permissions.

The process has an owner and is using resources.

Go up or down a directory.
You're in your home folder.

ssh to the box, machine, server.

Copy, delete, move, rename, link the files.
```
